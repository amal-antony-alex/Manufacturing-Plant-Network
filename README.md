This minor project focuses on the **design and simulation of a secure, scalable, and high-performance network infrastructure for a modern manufacturing plant**.  
The network is modeled using **Cisco Packet Tracer** and follows a **three-tier hierarchical architecture** consisting of Core, Distribution, and Access layers.

The design emphasizes **network segmentation, cybersecurity, redundancy, and future scalability**, aligning with **Industry 4.0** requirements.

- Simulate real-world enterprise and OT (Operational Technology) environments
